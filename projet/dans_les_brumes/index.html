>_ dans les brumes

site web qui présente la recherche de l’artiste-designer Camille Peyrachon_ 2022

Dans les brumes est le site web qui met en forme la recherche réalisée par l'artiste-designer Camille Peyrachon pour son mémoire de fin d'étude à l'ENSCI-les-Ateliers, Paris.
Elle réalise une enquête autour des savoir-faires tisserand dans le village de Saint-Julien-Molin-Molette en Auvergne-Rhône-Alpes.
