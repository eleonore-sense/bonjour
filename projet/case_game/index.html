éléonore sense

>_ case game & cie

série minis jeux vidéos qui utilisent un langage de programmation rudimentaire_ 2020/2023

Ces mini-projets font partie d’une série de recherches plastiques qui détourne les usages conventionnels des mises en page web en exploitant des zones inoccupées ou hors de leurs pratiques communes.
Le case game est mini jeu vidéo en ligne réalisé uniquement en bouton et champs de saisie de texte.
 jouer au case game.

L’encrypteur galactique a été réalisé avec la designeuse Jeanne Chiche.
Il joue sur la complémentarité du format numérique et impression.
Une personne reçoit une constellation. pour la décrypter, elle doit imprimer la page et ainsi faire apparaître le tableau décrypteur.
Différents indices se cache dans la page tel un lien vers la page Wikipédia sur l’imprimerie.
 envoyer un message crypté.
